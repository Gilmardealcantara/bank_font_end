# [BANK WEB APP]
#### WEB APP para simulação de transações bancárias.

## Pode-se:

* Listar Clientes
    * Tirar extrato
    * Realizar Transações simples (tansferência bancárias)
* Cadastrar clientes
* Listar Transações
* Lista Contas e seus respectivos saldos


## Tecnologias

Font-end javascript

* React
* Es6
* Bootstrap components
